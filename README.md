# EventFlow 🎉

A modern Event Booking web application built using React, Vite, React Router, and Tailwind CSS.

## Features

- Responsive UI
- Browse Events
- Event Details Page
- Categories
- Search UI
- Favorites Page
- Tickets Page
- Login Page
- Seat Selection UI

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Router
- JavaScript (ES6)

## Installation

```bash
npm install
npm run dev
